Robotics Studio Project

Project Overview

Bolto is a 3D-printed quadrupedal robot designed to achieve fast and stable walking, drawing inspiration from animals such as cats. The robot was developed as part of a studio course using a Raspberry Pi 4, 8 bus servo motors, and custom SolidWorks CAD. Bolto surpassed the required gait speed of 25 cm/s with a final walking speed of 33 cm/s.

Concept Development

Initial concepts included Techo Gecko (bipedal), Bob the Robot (quadruped), and Bolto the Dog (quadruped). Power consumption and runtime were estimated to be 13.8 minutes at full motor load using the 3000mAh battery.

Design & Inspiration

Bolto's leg design incorporates a spherical foot for smoother walking, lightweight cutouts for reduced material use, and arc-like cable guides for better wiring management. The design ensures balance by centralizing the heaviest component, the battery.

CAD & Electronics

CAD planning was key to housing electronic components, achieving clean wire routing, and ensuring proper motor positioning. The electronics include a Raspberry Pi connected via a BusLinker servo controller and a custom power distribution system.

Programming & Testing

Initial leg testing was done using sinusoidal motion patterns. Final code optimized these motions to achieve efficient walking. Simulation in PyBullet helped visualize gait dynamics, although some parameters were hand-tuned.

Outcome

Bolto achieved a walking speed of 33 cm/s, exceeding the project's benchmark. The project emphasized rapid prototyping, detailed CAD design, embedded programming, and systems integration—all within a two-person team.

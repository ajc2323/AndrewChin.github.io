Smart Bra Custom Test Jigs

To evaluate strain sensor performance, I designed a custom jig where one end of the sensor was zip-tied to a fixed metal base and the other to the retractable piston of a linear actuator. Care was taken to ensure the sensor remained taut without pre-strain. The linear actuator, powered and controlled by an Arduino Uno and motor driver, retracted at a constant 10 mm/s, elongating the sensor over time. A simple voltage divider circuit enabled real-time voltage readings across the sensor, captured via the Arduino’s analog pin and converted from raw values to voltages within the sketch. Simultaneously, elongation was calculated based on elapsed time and actuator velocity, providing a continuous mapping of strain to output voltage. Both datasets were logged in tandem for downstream analysis.

Pressure Jig Design & Protocol

To characterize pressure sensor performance, I developed a custom test jig featuring a linear actuator, syringe, and water receptacle to apply a controlled compressive load. As the actuator extends, water is displaced from the syringe into the receptacle, exerting a load on the pressure sensor; retraction removes the load. A load cell placed beneath the sensor serves as a ground truth reference, capturing the actual force applied. Using an Arduino Uno, I controlled the actuator’s motion and simultaneously logged real-time load cell data and pressure sensor voltage outputs. This setup enabled continuous cycling and synchronized data collection for downstream analysis.

Sensor Evaluation & Testing

I designed test protocols and circuits using Arduino microcontrollers to benchmark sensor behavior under compressive loading. Using amplifier and filter circuits, I quantified signal quality, sensitivity, and noise across sensor candidates. Data was collected using the serial monitor and visualized in real-time using Python plotting scripts.

Custom Test Jigs

I developed several custom jigs to evaluate sensor response to localized compression and stretch. These jigs featured modular inserts to accommodate various sensor sizes, applied loads using weighted sliders, and provided reliable test repeatability.

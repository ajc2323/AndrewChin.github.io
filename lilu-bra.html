Smart Sensing Bra Overview

As a product engineer at Lilu Inc., I worked on the early-stage development of a Smart Sensing Bra system intended to detect pressure and strain signals during lactation. This system was designed to collect real-time feedback to help guide massage functionality and improve user experience.

Sensor Research & Selection

I led an in-depth investigation into flexible pressure and strain sensors suited for wearable integration. Options included piezoresistive and capacitive films, textile-based sensors, and off-the-shelf modules. Key evaluation criteria included flexibility, repeatability, response time, and robustness during dynamic use.

Sensor Evaluation & Testing

I designed test protocols and circuits using Arduino microcontrollers to benchmark sensor behavior under compressive loading. Using amplifier and filter circuits, I quantified signal quality, sensitivity, and noise across sensor candidates. Data was collected using the serial monitor and visualized in real-time using Python plotting scripts.

Custom Test Jigs

I developed several custom jigs to evaluate sensor response to localized compression and stretch. These jigs featured modular inserts to accommodate various sensor sizes, applied loads using weighted sliders, and provided reliable test repeatability.

Integration Into Bra System

I translated electrical sensor layouts into physical prototypes, integrating selected sensors into textile samples. This included iterative prototyping with sewable contacts, signal routing, and comfort optimization for wearability. I also collaborated with the firmware team to align sensor outputs with massage logic and assisted in packaging layouts.
